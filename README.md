autex
========
texとかなんやかんやのコンパイルを自動化するためのスクリプトです。

Preparation
-----------------
任意のディレクトリでファイルが変更されたらそれをトリガーにしてなんかコマンドを実行するみたいなそんなやつです。
まずは準備をしましょう。

.. code-block:: bash

    cd /path/to/directory/that/in/your/PATH
    ln -s /path/to/autex.py autex

とかしとくとよい気がします。
